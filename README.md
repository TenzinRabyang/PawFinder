# PawFinder

PawFinder is a Next.js pet services directory app with location search, provider profiles,
review flows, Supabase integration, and AI-assisted provider analysis.

## Getting Started

Run the local development server:

```bash
npm run dev
```

Open `http://localhost:3000` in the browser.

## Deployment

The app is intended to be deployed on Vercel with the required environment variables
configured in the Vercel project settings.
